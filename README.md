# NGINX
Install:
    brew install nginx
    https://docs.nginx.com/nginx/admin-guide/installing-nginx/installing-nginx-open-source/

Edit Configs:
    subl /usr/local/etc/nginx/nginx.conf

Check Syntax:
    nginx -t

Reload Server:
    nginx -s reload

Start NGINX:
    nginx
